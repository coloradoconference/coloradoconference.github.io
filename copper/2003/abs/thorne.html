Cache Aware Multigrid on AMR Hierarchies

Danny Thorne

325 McVey Hall -- CCS
Lexington, KY 40506-0045
USA


Abstract

A cache optimized multilevel algorithm to solve variable coefficient elliptic boundary value problems on adaptively refined structured meshes is described here. The algorithm is optimized to exploit the cache memory subsystem. Numerical results are given demonstrating the efficiency of the cache optimization.
