Equivalent real formulations (ERFs) are useful for solving complex linear systems using real solvers. Using the four ERFs discussed by Day and Heroux [1], each can be expressed by multiplying the canonical form of the complex matrix by certain diagonal and permutation matrices on either side. This will allow, for instance, one ERF to be used as a preconditioner and another ERF to be used to iteratively solve the linear system by simply switching back and forth between the forms through scaling and permuting.
